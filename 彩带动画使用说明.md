# 彩带动画加载组件使用说明

## 概述

本项目已集成了彩带动画作为加载动画，可以在页面加载时显示美观的彩带效果。

## 组件说明

### 1. ConfettiLoader 组件

基础的彩带动画组件，支持不同尺寸。

```tsx
import { ConfettiLoader } from "@/components/ui/ConfettiLoader";

// 使用示例
<ConfettiLoader size="medium" />
```

**属性：**
- `size`: 'small' | 'medium' | 'large' - 动画尺寸（默认：large）
- `className`: string - 自定义样式类

### 2. LoadingWrapper 组件

加载包装器组件，可以在加载时显示彩带动画，加载完成后显示内容。

```tsx
import { LoadingWrapper } from "@/components/ui/LoadingWrapper";

// 使用示例
<LoadingWrapper loading={isLoading} size="medium">
  <div>页面内容</div>
</LoadingWrapper>
```

**属性：**
- `loading`: boolean - 是否显示加载状态
- `children`: React.ReactNode - 要显示的内容
- `size`: 'small' | 'medium' | 'large' - 动画尺寸（默认：large）
- `className`: string - 自定义样式类
- `fallback`: React.ReactNode - 自定义加载内容

### 3. ChaseLoader 组件

旋转点加载动画组件，适用于小范围的加载状态显示。

```tsx
import { ChaseLoader } from "@/components/ui/ChaseLoader";

// 使用示例
<ChaseLoader size="medium" />
```

**属性：**
- `size`: 'small' | 'medium' | 'large' - 动画尺寸（默认：medium）
- `className`: string - 自定义样式类
- `color`: string - 点的颜色（默认：#3b82f6）

## 自定义使用

### 在组件中使用

```tsx
import { useState, useEffect } from 'react';
import { LoadingWrapper } from "@/components/ui/LoadingWrapper";

function MyComponent() {
  const [isLoading, setIsLoading] = useState(true);

  useEffect(() => {
    // 模拟数据加载
    const timer = setTimeout(() => {
      setIsLoading(false);
    }, 2000);
    
    return () => clearTimeout(timer);
  }, []);

  return (
    <LoadingWrapper loading={isLoading} size="medium">
      <div>您的页面内容</div>
    </LoadingWrapper>
  );
}
```

### 自定义加载内容

```tsx
<LoadingWrapper 
  loading={isLoading} 
  size="large"
  fallback={<div>自定义加载内容</div>}
>
  <div>页面内容</div>
</LoadingWrapper>
```

## 技术实现

- **ConfettiLoader**: 使用 `lottie-web` 库播放彩带动画
- **ChaseLoader**: 使用纯CSS实现旋转点动画
- 动画文件位置：`public/confetti-animation.json`
- 支持自动播放和循环
- 响应式设计，适配不同屏幕尺寸
- **ConfettiLoader尺寸**: small(128px) | medium(192px) | large(256px)
- **ChaseLoader尺寸**: small(24px) | medium(40px) | large(64px)

## 注意事项

1. 确保 `lottie-web` 依赖已安装
2. 动画文件必须放在 `public` 目录下
3. 加载时间可以根据实际需求调整
4. 建议在数据加载完成后才隐藏加载动画
